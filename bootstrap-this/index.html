Intro
Bootstrap is a mobile-first grid type structure used in HTML, CSS, and JS. It uses a responsive grid system along with built in components for layout and styling
How it works (basics)
Bootstrap uses a layout in which container classes are used as the basic layout.setup. Inside a container wouldbe a tag with a class of row, and inside there is where we would set up our column layout. All content is placed specifically inside the columns.
Breakpoints
There are breakpoint abbreviations; sm, md, lg, xl. Using these abbreviations by simply writing them after the 'col' element (ex: div class="col-lg-3")(this is saying when the page reaches a breakpoint at size large, the column will take up 3 spaces out of a total of 12 which will help us set up some breakpoints, with each abbreviation containing their own size (in pixels).
Alignment
Vertical We can use 'align-items' with the values of 'start', 'end', 'center', 'baseline', and 'stretch'. Horizontal We can use 'justify-content' with the same values.
Offset
'offset-' followed by a number from 1-11 can be used to move the columns to the right for a desired alignment/layout.
Reordering
We can use the '.order' class to order the items in order prefered. Values that can be used include '-first', 'last'. and any number from '1-12'.
